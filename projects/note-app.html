Overview

A clean and minimal note-taking application with markdown support, tagging system, and cloud synchronization for seamless access across devices.

Key Features

  • Rich markdown editor with live preview
  • Tagging and categorization system
  • Full-text search capabilities
  • Cloud sync across devices
  • Dark mode support
  • Export to PDF and Markdown

Technologies Used

  • Frontend: React, CodeMirror
  • Backend: Firebase
  • Storage: Firebase Cloud Storage
  • Authentication: Firebase Auth

Outcome

Created a distraction-free note-taking experience that helps users capture and organize their thoughts efficiently.
